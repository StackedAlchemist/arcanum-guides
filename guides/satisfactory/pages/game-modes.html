Game Modes

Update 1.2 splits world customization into two systems: Game Modes, which let you bend cost and resource rules without losing achievements, and Creative Mode, the full sandbox that does. Here is exactly what every slider does and which combinations make the run you actually want.

Two Systems Cost Multipliers World Randomization Creative Mode Preset Builds Rules & Gotchas
Read this first

Game Mode settings are chosen only when you create a New Game, and once a save has Game Mode settings active they cannot be turned off for that save. They also apply to everyone in a multiplayer session. Decide before you commit — there is no in-game toggle to revert later.

Two systems, one menu

Before 1.2, all of this lived under "Advanced Game Settings," which permanently disabled achievements the moment you touched it. Update 1.2 rebuilt that:

The practical takeaway: if you want a legit run that's just easier or harder than default, use Game Modes. If you want a god-mode sandbox to test layouts or build art, use Creative Mode and accept the lost achievements.

Cost multipliers

Three independent sliders. Each scales a different part of the economy. Defaults are all 1.0×.

MultiplierRangeWhat it changesLower it to…Raise it to…
Space Elevator Deliverable Cost 0.25× – 100× How many parts each Space Elevator phase / Project Assembly delivery demands. Blast through phases and reach end-tier buildings fast. Turn the whole game into a megabase marathon (up to 100× is a true endurance run).
Recipe Parts Cost 0.25× – 2× Ingredient cost of crafting recipes — how much input each machine consumes per product. Stretch raw nodes much further; smaller factories hit the same output. Force tighter logistics and bigger mines (2× is a real squeeze).
Power Consumption 0.25× – 5× How much power every building draws. Make power a non-issue and focus purely on production. Make the power grid the central puzzle (5× punishes overclocking hard).
Slider interaction

These stack with overclocking and Somersloop production-doubling. At 5× power, an overclocked-to-250% machine becomes brutally expensive, because overclock power cost scales exponentially (^1.321). If you crank Power Consumption, plan to run machines at or below 100% and just build wider.

World randomization

Two settings reshape the map itself, plus a world seed. This is the part that makes a fresh playthrough feel genuinely new instead of "the same Grass Fields start again."

Resource node distribution

Controls which resource sits in each node slot across the map:

  • Default — the hand-authored map you already know.
  • Random — node resources shuffled. The classic randomizer experience; you may have to go hunting for your first coal.
  • Basic Resource Rich — weighted toward iron, copper, limestone, caterium. Friendly early game, great for new players.
  • Advanced Resource Rich — weighted toward late-game ores (bauxite, uranium, SAM, quartz). A "skip-the-grind" endgame sandbox.
  • Fossil Fuel Rich — weighted toward coal and crude oil. Power and plastics/rubber are trivial; ideal for a fuel-power-focused build.

World seed

You can enter a specific world seed to share a randomized layout with friends or replay a known map. Two saves with the same seed and the same randomization settings generate the same node placement — handy for community challenges and races.

Creative Mode

This is the old Advanced Game Settings, renamed and kept fully featured. Toggling it on disables achievements for that save, permanently. What it unlocks:

  • No build cost — place anything for free.
  • No power requirement — machines run with no grid.
  • Unlimited / give items — spawn parts directly.
  • Flight mode & god mode — fly the map, take no damage.
  • Unlock all research / tiers / schematics — skip the MAM and milestone grind.
  • Time of day control & the new weather presets.

Use it as a blueprint lab: prototype a factory layout at full scale for free, snapshot it as a Blueprint, then rebuild it legitimately in your real save.

Rules & gotchas

  • Game Modes keep achievements; Creative Mode kills them. If a Steam/Epic completion matters to you, never enable Creative on that save.
  • Permanent once active. Game Mode multipliers and randomization are locked in at world creation. Plan the run you want before clicking Create.
  • Multiplayer-wide. The host's Game Mode settings apply to every player in the session.
  • Randomization changes starting logistics. With Random resources you may not have coal near spawn — keep biomass burners going longer and scout before committing a base location. See Getting Started.
  • Cost multipliers don't change recipes, only quantities. Every ratio on the Ratios page still holds; you're just feeding more or fewer raws into the same machine counts.
